# Workshop-DIY — Générateur de Posts V6.0

Outil multi-mode de création de visuels pour **Workshop-DIY** (Chelles 77500). Application 100% client-side, fichier HTML unique, zéro dépendance serveur. PWA installable.

## 5 Modes de Création

### 🎨 Visuels — 58 Styles
Le mode principal. Génère des visuels Facebook avec texte, logo, doodles et footer.
5 catégories: Classique (15), Kiddy (10), Formes (10), Andalous (13), Intense (10).
Thumbnails live, favoris ⭐, historique Préc/Suiv (30 posts), style auto/manuel.

### 🐄 Cowsay — 35 Animaux × 16 Fonds
ASCII art en image. 35 animaux, 16 arrière-plans, 3 bulles (dire/penser/crier), 6 formats (terminal, git, code, BSOD, fortune 50+50 citations FR/AR), modes solo/duo/custom/combo.

### 💬 Chat / SMS — 6 Styles
Fausses conversations: iMessage, WhatsApp, Messenger, SMS, Discord, Telegram. Max 8 bulles, personnages personnalisables, cadre iPhone, timestamps. Contenu FR/AR.

### 😂 Mème — 12 Templates
Drake, Galaxy Brain, Distracted, Bouton Rouge, Scroll, Expanding, Change My Mind, This is Fine, Success Kid, Aliens Guy, Two Buttons, Stonks. Upload image perso. Textes rapides FR/AR.

### 📱 Notification — 6 Styles OS
iOS Light/Dark, Android Light/Dark, Windows 11, macOS. 12 icônes, textes rapides FR/AR.

## Fonctionnalités

### Live Preview
Tous les champs dans tous les modes régénèrent l'image en temps réel.

### Bilingue FR / 🇩🇿 AR
- 30 messages FR + 30 messages AR (فصحى)
- 50 fortune cookies FR + 50 AR (حكم وأمثال)
- Quick texts FR/AR dans les 5 modes
- Placeholders arabes quand lang=AR
- 4 polices arabes: Cairo, Amiri (calligraphie), Tajawal (moderne), Noto Kufi (géométrique)
- Attribution auto par style: Amiri pour Andalous, Tajawal pour Kiddy
- RTL complet dans tous les modes

### Texte Libre (✏️)
Texte draggable sur l'image. 9 polices (dont 3 arabes), couleur/taille, multi-textes, undo/clear.

### Galerie & Export/Import
- 🖼️ Galerie visuelle (thumbnails 200px, max 100, persistée)
- 📥 Export JSON (settings + favoris + galerie + compteur + QR)
- 📤 Import JSON (restauration entre machines, fusion sans doublons)
- Compteur total de posts générés
- 🗑 Reset données

### PWA / Installation
Manifest intégré, Service Worker cache-first, bouton 📲. Fonctionne offline.

### Raccourcis Clavier
Ctrl+Enter (générer), Ctrl+S (PNG), Ctrl+Shift+R (aléatoire).

### Autres
- 3 formats: Carré 1080×1080, Paysage 1200×630, Story 1080×1920
- 4 sources: Intégré, Manuel, Groq (IA gratuite), Claude (IA Anthropic)
- QR Code drag & drop, URL/couleur/taille personnalisables
- Batch export par mode (58+35+12+6+6)
- Favoris ⭐ persistés, aide modal ❓, thème clair/sombre
