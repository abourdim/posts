GÉNÉRATEUR DE POSTS V3.0

Créez des visuels inspirants pour votre page Facebook

❓ Guide Rapide

1. Choisissez un mode dans les onglets du haut (🎨 🐄 💬 😂 📱)
2. Remplissez le message ou sujet
3. Choisissez un style dans la grille
4. Cliquez GÉNÉRER
5. Téléchargez avec ⬇ PNG ou partagez avec 📤

Astuce: Cliquer sur un style/animal/template régénère automatiquement l'image!

Formats: ⚙️ Paramètres → Carré (Facebook), Paysage (lien), Story (Instagram/TikTok)

🎨 Visuels — 58 styles professionnels. Filtrez par catégorie, marquez vos ⭐ favoris. Historique avec Préc/Suiv.

🐄 Cowsay — 25 animaux ASCII × 16 fonds. Modes: Solo, Duo (2 animaux), ASCII perso, Combo visuel. Bulles: dire, penser, crier. Formats: terminal, git, code, BSOD, fortune.

💬 Chat — Fausses conversations iMessage/WhatsApp/Messenger/SMS/Discord/Telegram. Max 8 bulles. Cadre téléphone optionnel.

😂 Mème — 8 templates classiques + 8 textes rapides pré-écrits. Texte Impact avec outline.

📱 Notif — 6 styles OS (iOS, Android, Windows, macOS) + 12 icônes + 6 textes rapides. Réaliste avec barre d'état et ombre.

⬇ PNG — Télécharge l'image (nom inclut le mode et style)

📤 Partager — Partage direct sur mobile (WhatsApp, Facebook...) ou copie image sur desktop

📋 Texte — Copie le texte + contacts pour coller sur Facebook

↻ — Régénère avec les mêmes paramètres

🎲 — Style/animal/template aléatoire

📦 — Export batch: toutes les variantes du mode actif

QR — Télécharge le QR code seul (visible quand QR activé)

Batch par mode: Visuels=58 images, Cowsay=25, Chat=6, Mème=8, Notif=6

Activer l'IA: ⚙️ Paramètres → Source → Groq ou Claude

Groq est gratuit: Créez un compte sur groq.com/console et copiez votre clé API (commence par gsk_)

Claude: Nécessite une clé API Anthropic (commence par sk-ant-)

En mode Visuels: L'IA génère automatiquement le texte du post.

Autres modes: Le bouton ✨ IA Suggérer apparaît dans la barre latérale. Il génère un message cowsay, une conversation chat, un texte de mème, ou une notification.

Mes réglages sont-ils sauvegardés?
Oui! Tout est persisté en localStorage: thème, langue, format, favoris, dernière position QR.

Puis-je utiliser l'outil hors ligne?
Oui, sauf les modes IA et le QR personnalisé.

Comment poster sur Facebook?
1) Téléchargez le PNG 2) Uploadez dans votre post Facebook 3) Collez le texte avec 📋 Texte

Le logo ne s'affiche pas?
Le logo est chargé depuis internet. Hors ligne, il est masqué automatiquement.

Comment ajouter mes propres styles?
Les styles sont définis dans le JavaScript. Ajoutez une entrée dans STYLES et un renderer dans RENDERERS.

Format Story?
⚙️ Paramètres → Format → Story (1080×1920). Idéal pour Instagram et TikTok.

Batch télécharge trop de fichiers?
Votre navigateur peut demander l'autorisation pour les téléchargements multiples. Autorisez-le.
