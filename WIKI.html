Workshop-DIY — Guide d'Utilisation V6.0

Démarrage Rapide

1. Ouvrez workshop-post-gen.html (ou installez via 📲)

2. Choisissez un mode: 🎨 🐄 💬 😂 📱

3. Modifiez le texte — l'image se met à jour en temps réel

4. ⬇ PNG pour télécharger ou 📤 pour partager

Modes

🎨 Visuels

Sujet → style (58 choix ou auto) → favoris ⭐ → historique Préc/Suiv.

🐄 Cowsay

Message → animal (35) → fond (16) → bulle/format. Mode Duo, Custom ASCII, Combo.

💬 Chat

Bulles auto-fill (4). Ajouter/supprimer/swap. Noms et avatars personnalisables. Cadre iPhone, timestamps.

😂 Mème

Texte haut/bas (live). 12 templates ou image uploadée (📷). Textes rapides.

📱 Notif

Titre/message/sous-texte (live). 6 styles OS, 12 icônes. Textes rapides.

Langue Arabe 🇩🇿

Basculez via ⚙️ → 🇩🇿 العربية. Tout change automatiquement:

  • Messages et textes rapides en arabe dans les 5 modes
  • Placeholders arabes dans tous les champs
  • Fortune cookies: 50 حكم وأمثال
  • Chat: bulles et noms en arabe (كريم/ورشة)
  • 4 polices: Cairo, Amiri (calligraphie), Tajawal (moderne), Noto Kufi (géométrique)
  • Polices auto selon le style: Amiri pour Andalous, Tajawal pour Kiddy
  • RTL complet

Texte Libre (✏️)

1. Cliquez ✏️ dans la barre d'actions

2. Tapez texte, choisissez police (9 dont 3 arabes)/couleur/taille

3. Entrée ou "Appliquer"

4. Glissez pour positionner. ↩ undo, 🗑 clear.

Galerie & Transfert

Galerie

⚙️ → 🖼️ Galerie. Thumbnails avec mode/style/date. Supprimer au survol.

Transférer entre machines

1. ⚙️ → 📥 Exporter JSON

2. Transférer le fichier

3. ⚙️ → 📤 Importer JSON

Fusionne sans doublons. Restaure favoris, galerie, paramètres, compteur.

Paramètres (⚙️)

  • Format: Carré / Paysage / Story
  • Langue: 🇫🇷 Français / 🇩🇿 العربية
  • Source: Intégré / Manuel / Groq / Claude
  • Données: Export/Import JSON, Galerie, Reset

Raccourcis

  • Ctrl+Enter: Générer
  • Ctrl+S: Télécharger PNG
  • Ctrl+Shift+R: Aléatoire

Actions

⬇ PNG · 📤 Partager · 📋 Texte · ↻ Régénérer · 🎲 Aléatoire · ✏️ Texte libre · 📦 Batch · QR

FAQ

Ça marche hors ligne? Oui via PWA (sauf IA et polices Google).

Comment transférer entre machines? Export/Import JSON dans ⚙️.

Le batch export est lent? Normal, ~30s pour 58 images.

Image perso pour mème? Mode Mème → 📷 Uploader une image.

Polices arabes? Auto selon style. Amiri pour calligraphie, Tajawal pour enfants, Noto Kufi géométrique.
